# Phase 3 Screenshot Capture Checklist

Use this checklist when exporting dashboard screenshots for portfolio publishing.

## Required Files

- `01-executive-dashboard.png`
- `02-supplier-performance.png`
- `03-savings-opportunities.png`
- `04-risk-analysis.png`

## Capture Rules

- Resolution: 1920x1080 minimum
- Theme: single consistent theme across all screenshots
- File type: PNG only
- Avoid personal identifiers or machine-specific metadata in visuals
- Keep visual labels legible at 100% zoom

## Content Requirements

## 01 Executive Dashboard

- KPI cards: total spend, total savings, savings %, maverick spend
- Category spend visual
- Monthly spend trend

## 02 Supplier Performance

- Supplier table with OTD, quality, risk indicators
- Underperformer ranking visual
- Quality/delivery comparison visual

## 03 Savings Opportunities

- Price variance top opportunities
- Optimization recommendations
- Category/material drilldown visual

## 04 Risk Analysis

- Maverick spend view
- FX exposure/volatility summary
- Monte Carlo confidence interval display (P05/Median/P95)

## Signoff

- [ ] All 4 images exported with exact names
- [ ] Images saved under `docs/screenshots`
- [ ] README links validated
- [ ] Portfolio narrative updated
