# Power BI Capture Guide

Use this guide when exporting final Power BI screenshots after building the report from the deployment pack.

## Recommended Power BI screenshot set

1. `powerbi-01-executive-overview.png`
2. `powerbi-02-supplier-performance.png`
3. `powerbi-03-savings-opportunities.png`
4. `powerbi-04-risk-uncertainty.png`
5. `powerbi-05-data-quality.png`

## Capture standards

- Canvas: 16:9
- Resolution: 1920x1080 minimum
- Theme: apply `powerbi_theme.json`
- Export type: PNG
- Keep slicers visible when they add context

## Page mapping

- Executive Overview
- Supplier Performance
- Savings Opportunities
- Risk & Uncertainty
- Data Quality & Refresh Notes

## What to show

### Executive Overview

- KPI cards
- category spend bar chart
- monthly trend line chart
- scenario comparison visual

### Supplier Performance

- supplier matrix
- OTD vs quality scatter
- top suppliers by spend

### Savings Opportunities

- top price variance bars
- optimization table
- constrained sourcing plan

### Risk & Uncertainty

- maverick spend by risk level
- FX cards
- Monte Carlo table

### Data Quality & Refresh Notes

- source label
- upload or refresh notes
- diagnostics table

## Portfolio recommendation

Use Streamlit screenshots as the main story and Power BI screenshots as proof of BI deployment readiness.
