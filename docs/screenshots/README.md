# Screenshot Guide

Add production-quality dashboard screenshots to this folder using the file names below:

1. 01-executive-dashboard.png
2. 02-supplier-performance.png
3. 03-savings-opportunities.png
4. 04-risk-analysis.png

## Capture Standards

- Resolution: 1920x1080 or higher.
- Theme: consistent light or dark theme across all images.
- Export format: PNG.
- Keep KPI cards and chart labels readable at 100% zoom.
- Remove personal or confidential identifiers before export.

## Recommended Content per Screenshot

### 01-executive-dashboard.png

- Total spend, savings potential, maverick spend, FX exposure KPIs.
- Category spend distribution chart.
- Monthly spend trend chart.

### 02-supplier-performance.png

- Supplier scorecard table.
- On-time delivery vs quality cost scatter or matrix.
- Top poor performers bar chart.

### 03-savings-opportunities.png

- Top materials by price variance.
- Savings waterfall or ranked bars.
- Action-priority list.

### 04-risk-analysis.png

- Maverick buying split and trend.
- FX volatility trend.
- High-risk suppliers summary.
